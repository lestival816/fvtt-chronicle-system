# Chronicle System for Foundry VTT (Unofficial)

This is an implementation for the **Chronicle System** by **Green Ronin**. This system is used by _Sword Chronicle: Feudal Fantasy Roleplaying_ and _A Song of Ice & Fire Roleplaying_. 

The implementation code is based on the Boilerplate System by Asacolips.

This is a fan-created system implementation. The Chronicle System is trademark of Green Ronin Publishing, LLC in the U.S.A. and other countries. 

## Features

- Character sheet support (playbale character and non playable character sheets)
- Combat tracker with combat initiative
- Dice rolls with custom modifiers 
- Fatigue, Frustration, Injury, and Wound values affect the character's rolls

### Know Limitations

- No Sorcery rules support
- Armor Penalty to be set manually
- Only Human Ancestry

New features are expected to be added over time.
